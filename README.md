# AverageFinder
Finds the average of the numbers in an array.  

## How to compile  
1. Open the Command Prompt  
2. ``cd`` to the folder in which you unziped the application  
3. Run ``javac *.java``  

## How to run
1. Open the Command Prompt
2. ``cd`` to the folder in which the application is in  
3. Run ``java Main``
